# RobieLens

## Introduction

Bienvenue dans le dépôt GitHub de Robbie Lens Photographie ! Ce projet est l'aboutissement du [cours de HTML/CSS](https://openclassrooms.com/fr/courses/1603881-creez-votre-site-web-avec-html5-et-css3) proposé par OpenClassrooms. Il  présente le portfolio de Robbie Lens, une photographe talentueuse avec plus de 5 ans d'expérience capturant l'essence de chaque moment. Le dépôt contient des fichiers HTML et CSS, ainsi que des images organisées de manière structurée.

## Structure du Projet

### Fichiers HTML

1. **index.html**
   - Page d'accueil présentant Robbie Lens Photographie.
   - Liens de navigation vers "Accueil", "À propos", et "Portfolio".
   - Sections incluant une introduction, des photos en vedette, et un formulaire de contact.

2. **a-propos.html**
   - Page "À propos" détaillant les services photographiques de Robbie Lens.
   - Sections couvrant le parcours du photographe, les services proposés, et les détails tarifaires.

3. **portfolio.html**
   - Page Portfolio mettant en valeur le travail photographique de Robbie Lens.
   - Sections affichant des photos de paysages et de portraits avec des effets de survol interactifs.

### Fichiers CSS

1. **style.css**
   - Styles globaux pour l'ensemble du projet.
   - Définit les polices, les couleurs et les styles communs utilisés à travers les pages.

2. **index.css**
   - Styles supplémentaires spécifiques à la page d'accueil (`index.html`).
   - Définit les styles pour les sections d'introduction, de photos en vedette, et de formulaire de contact.

3. **a-propos.css**
   - Styles pour la page "À propos" (`a-propos.html`).
   - Définit la mise en page des blocs de contenu, de la section des services et du tableau tarifaire.

4. **portfolio.css**
   - Styles pour la page Portfolio (`portfolio.html`).
   - Définit la mise en page des grilles de photos, des effets de survol et de la réactivité pour différentes tailles d'écran.

## Outils Utilisés

Ce projet utilise plusieurs outils de développement web, dont certains sont notables :

- **Flexbox :** Utilisé pour créer des mises en page flexibles et réactives.
- **Grid :** Utilisé pour organiser les photos du portfolio dans des grilles.
- **Media Queries :** Utilisées pour rendre le site adaptatif à différentes tailles d'écrans.

## Comment Utiliser

1. Clonez le dépôt : `git clone https://github.com/votre-nom/robbie-lens-photographie.git`
2. Ouvrez les fichiers HTML dans votre navigateur web préféré pour afficher les différentes pages.

N'hésitez pas à explorer le code, modifier le contenu et personnaliser les styles selon vos besoins. Si vous avez des questions ou des suggestions, n'hésitez pas à me contacter !
